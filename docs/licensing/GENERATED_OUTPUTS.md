# Generated Outputs

> Status: DRAFT — generator-by-generator classification is still required.

Generated files are classified by what the generator copies and by the terms
of its templates and inputs.

## Approved default for minimal scaffolding

Minimal, non-expressive project scaffolding and approved gameplay boilerplate
may be released under 0BSD when the generator explicitly declares that output
policy.

## No automatic exception

The following are not automatically 0BSD:

- copied Saga implementation;
- generated amalgamations containing MPL-covered files;
- third-party code or assets;
- generated documentation derived from restricted inputs;
- user content for which Saga has no licensing authority.

Each generator must emit or preserve provenance and license metadata.
