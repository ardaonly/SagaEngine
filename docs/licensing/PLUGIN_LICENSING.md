# Plugin Licensing

> Status: DRAFT — general repository policy, not legal advice.

Independent plugin files may use a license selected by their rights holder.
Using public Saga APIs does not automatically place those independent files
under MPL-2.0.

MPL obligations may apply when a plugin distribution modifies, copies, or
incorporates MPL-covered Saga files. Third-party dependencies retain their own
terms. Plugin packaging must preserve required notices and source obligations.
