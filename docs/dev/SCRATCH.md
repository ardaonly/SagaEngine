# Dev Scratch

Engine version: 0.0.9

## Added

- Added Apps/SagaEditorComposer executable.
- Composer can read .sde editor composition source.
- Composer can show the composition tree and inspector.
- Composer can edit displayName and panel defaultVisible.
- Composer creates checkpoints when saving .sde files.
- Composer can run builds through SagaPipeline.
- Added Safe Customize Workspace UI.
- Added Safe Shortcut Preferences UI.
- Added SagaPipeline orchestration layer.
- SagaEditorComposition compiler can be produced as an executable in the SDE-enabled profile.
- Added xcb static plugin support for Qt visible launch.
