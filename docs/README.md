# SagaEngine Documentation

This directory explains what SagaEngine is today, how to inspect it safely, and
which documents are current versus historical.

SagaEngine is a long-term engine and toolchain project. It has real engine,
server, editor, tool, and test code, but it is not a finished game engine
product or a public SDK release.

## Main Areas

- [Product docs](product/) describe the current product identity and limits.
- [Architecture docs](architecture/README.md) describe current subsystem design
  and link to deeper technical notes.
- [Roadmaps](roadmaps/README.md) describe direction and historical planning
  context.
- [Testing docs](testing/README.md) describe safe verification entry points.
- [Development notes](dev/README.md) contain internal repo accounting and
  iteration history.
- [Internal docs](internal/README.md) preserve historical product and recovery
  material that should not be read as current product documentation.

## Reading Rule

Prefer the short product and architecture entry points first. Older files with
numbered planning or audit-style names are historical records unless a current
index points to them as active work.
