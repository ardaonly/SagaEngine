# SagaEngine Documentation

This directory explains what SagaEngine is today and where to find current
technical details.

SagaEngine has real engine, server, editor, tool, and test code. It is still an
engine/toolchain codebase, not a finished game engine product or public SDK.

## Start Here

- [What Is SagaEngine?](product/WHAT_IS_SAGAENGINE.md)
- [Current Capabilities](product/CURRENT_CAPABILITIES.md)
- [What Is Not Implemented](product/WHAT_IS_NOT_IMPLEMENTED.md)
- [Getting Started](product/GETTING_STARTED.md)
- [Current Distribution Status](product/CURRENT_DISTRIBUTION_STATUS.md)

## Main Areas

- [Product docs](product/) describe what works, what is missing, and how to
  start.
- [Architecture docs](architecture/README.md) summarize current subsystem
  boundaries.
- [Testing docs](testing/README.md) describe local verification entry points.
- [Development notes](dev/README.md) and [internal docs](internal/README.md)
  preserve repo history and should not be used as product onboarding material.

## Reading Rule

Prefer the short product and architecture entry points first. Files labeled as
appendices, proposals, history, numbered milestones, or audits are background
unless a current index points to them for a specific reason.
