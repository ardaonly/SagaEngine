# Small-Team Alpha C# + Blocks Authoring Guide

Status: Phase 94 onboarding evidence for bounded C# + Blocks authoring.

Small-Team Alpha authoring uses C# source as the canonical source truth. Visual
block evidence is limited to supported projection and source-preserving patch
reports.

## Supported Evidence

- C# behavior projection report.
- Supported patch diff report.
- Patch review report.
- Script artifact validation report.
- Runtime-proof classification for `ProjectionOnly` and `Deferred` nodes.

Review approval in collaboration metadata is not patch application. SagaScript
patch behavior remains separately invoked and separately reported.

## Unsupported Workflows

- arbitrary C# conversion into blocks;
- full visual scripting;
- editor-owned C# writes;
- graph editing as runtime proof;
- Runtime or Server gameplay expansion.

## Non-Claims

- no product beta
- no release candidate
- no production MMO server
- no complete visual scripting
- no arbitrary C# roundtrip
- no enterprise readiness
