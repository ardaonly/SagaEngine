## Phase Scope

C# Compatibility Profile v1

Phase 13 defines the first Saga-compatible C# profile boundary for future
Visual Blocks work. It adds documentation and SagaScript compatibility-profile
fixtures for:

- projectable C#;
- partially projectable C#;
- advanced opaque C#;
- unsupported C#.

Accepted boundary:

- C# remains canonical source;
- compatibility evidence is generated by `sagascript compatibility-profile`;
- fixtures assert source bytes are not mutated;
- advanced C# remains opaque/read-only;
- unsupported source is surfaced explicitly;
- no Visual Blocks UI, source patching implementation, runtime gameplay change,
  server change, editor workflow, package output, distribution output, or SDE
  work is included.
