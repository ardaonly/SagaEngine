# Phase Evidence

Internal evidence folders live here. These files are not public onboarding
material and must not be used as marketing claims.

Each phase folder records current evidence, command logs, changed files, known
limitations, and a machine-readable verification result. A phase folder existing
does not mean the phase is complete.

Allowed Codex-authored statuses are:

```txt
Not Started
In Progress
Implemented-Unverified
Blocked
```

`Verified` requires maintainer review or an explicit verification script result
accepted by maintainers.
